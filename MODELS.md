- 成本控制：★★★★★

## 如何选择模型？

### 选择 deepseek-v4-flash 如果：
- ✅ 需要快速响应
- ✅ 处理简单到中等复杂度的任务
- ✅ 关注成本控制
- ✅ 进行日常编程辅助

### 选择 deepseek-v4-pro 如果：
- ✅ 需要最高质量的回答
- ✅ 处理复杂的代码任务
- ✅ 进行系统设计和架构咨询
- ✅ 需要专业的代码审查

### 不推荐使用：
- ❌ deepseek-chat（即将弃用）
- ❌ deepseek-reasoner（即将弃用）

```
简单问题 → v4-flash（快速、经济）
```

- 设置合理的 `Max Tokens`（避免浪费）
- 使用适当的 `Temperature`（控制创造性）

## 迁移指南

如果您正在使用即将弃用的模型，建议尽快迁移：

### 从 deepseek-chat 迁移
1. 打开设置：`Settings -> Tools -> DeepSeek AI`
2. 将模型从 `deepseek-chat` 改为 `deepseek-v4-flash` 或 `deepseek-v4-pro`
3. 保存设置
4. 测试新模型的效果

### 从 deepseek-reasoner 迁移
1. 打开设置：`Settings -> Tools -> DeepSeek AI`
2. 将模型改为 `deepseek-v4-pro`（推荐，推理能力强）
3. 保存设置
4. 对比新旧模型的推理能力

## 常见问题

**Q: v4-flash 和 v4-pro 的主要区别是什么？**
A: v4-flash 速度快、成本低，适合日常使用；v4-pro 质量更高，适合复杂任务。

**Q: 为什么 chat 和 reasoner 要弃用？**
A: DeepSeek 推出了更先进的 v4 系列模型，性能和效率都有显著提升。

**Q: 弃用后还能继续使用吗？**
A: 弃用日期后可能无法调用，建议提前迁移到 v4 系列。

**Q: 可以随时切换模型吗？**
A: 可以！在设置中随时切换，立即生效。
